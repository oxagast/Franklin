     What wsl versions are there?

There are currently two versions of Windows Subsystem for Linux (WSL), WSL 1 and WSL 2. WSL 1 is based on a Linux 4.4 kernel and provides a full Linux command-line experience. WSL 2 is based on a Linux 4.19 kernel and offers improved system call compatibility and faster file system performance.
