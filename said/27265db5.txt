Citrix is a software used to provide virtual access to applications and data over a network. It is used for remote access, server and application virtualization, application streaming, as well as desktop virtualization. It is used by organizations to provide secure access to applications and data from any device, while reducing costs associated with managing multiple desktops, servers, and applications.
