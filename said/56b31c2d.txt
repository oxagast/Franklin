A plank is an exercise that strengthens the core and helps to build endurance and stability. It is done by placing your body in a straight line and holding it in the same position for an extended period of time. It can be done with the arms extended or in a push-up position. Planks can be done in a variety of ways, which makes it a great exercise for any fitness level.
